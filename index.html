SourcePanel

A free and open-source control panel designed to run Source engine dedicated servers






Web Based

Servers can be controlled from any device with a web browser. Linux, Mac, Windows, you name it. The control panel is completely web based.

Mobile Friendly

The control panel is mobile friendly and allows mobile users to easily control their server. Supports most popular mobile web browsers.

Material Design

The entire website/control panel design is based on Google's material design. The site/control panel is powered by Materialize CSS.

MIT License

The control panel/website is licensed under MIT so you can use the source code for practically whatever you want. Enjoy it if you want.
